Tentang Saya

Salam kenal, saya Nailah Adyan Nurahmah kerap dipanggil Nailah, seorang mahasiswi semester 2 program studi Teknologi Informasi di Politeknik Negeri Madiun. Dengan latar belakang pendidikan terakhir di MAN 1 Kota Madiun yang bekerja sama dengan Prodistik Intitut Teknologi Sepuluh Nopember alias ITS. Dari situ akhirnya saya memutuskan untuk melanjutkan studi saya di PNM dan mengambil program studi Teknologi Informasi.


Sebenarnya PNM dan Prodi Teknologi Informasi bukanlah tujuan utama saya dalam melanjutkan studi. Minat saya adalah di bidang yang berkaitan dengan biologi yaitu Ilmu Lingkungan namun, selain gagal di SNBP dan SNBT akhirnya saya memutuskan untuk mendaftar jalur prestasi dengan berbekal nilai rapot dan sertifikat kejuaraan olimpiade biologi serta tugas akhir prodistik ITS saya. Dan akhirnya saya LOLOS di PNM dengan pilihan pertama yaitu Teknologi Informasi.


Selain gagal di SNBP dan SNBT, orang tua saya terutama Alm papa tidak mengizinkan saya untuk kuliah diluar kota dan menghendaki saya untuk kuliah di PNM saja. Namun pada saat itu saya tetap nekat untuk mendaftar SNBP pil.1 Ilmu Lingkungan di UNS dan merelakan pil.2 alias saya kosongkan. Tetapi hasilnya merah wkwk. Setelah gagal di SNBP saya pun mencoba untuk mendaftar SNBT dengan pil.1 Teknologi Informasi PNM dan pil.2 Ilmu Lingkungan UNS, namun hasilnya tetap merah juga. Dan pada akhirnya saya mencoba jalur prestasi di PNM hingga akhirnya pada saat ini saya berkuliah di PNM.


FYI restu orang tua itu penting ya teman-teman!!!
